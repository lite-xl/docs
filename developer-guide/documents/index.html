    Documents

    Lite XL stores an opened file in a Doc. A Doc contains the file in lines (doc.lines), metadata and various states, including the syntax highlighter (doc.highlighter) and the undo stack.

    Creating Docs

    To create a Doc, you usually create instances of it directly. Instead, it is often easier to call core.open_doc(). This function will do bookkeeping and various checks before creating the Doc with the correct state.

    function core.open_doc(filename: string): Doc end

    The function accepts a filename and returns a Doc.
